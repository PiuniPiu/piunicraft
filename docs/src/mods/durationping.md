# Plugin: Duration Ping

<br/>

## Descripción
---

> Este plugin te permitirá establecer una notificación que te avisará de que la duración de la herramienta que tienes está por debajo de un %.

<br/>

## Comandos
---

<br/>

### Abrir la configuración (aparece en el chat)

```
/trigger duraPing
```
