# Plugin: Admin Gui

<br/>

## Descripción

---

> Este plugin generará una interfaz con muchas opciones para los administradores

<br/>

## Comandos

---

<br/>

> Este plugin no tiene comandos disponibles para el jugador normal, pero sí para los administradores.

<br/>

### Entrar en la interfaz

```
/admin
```
